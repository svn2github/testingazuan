Introduction to SpagoBI Platform 1.9.3 installation procedure

This installation file will guide you during the SpagoBI Platform installation steps. The procedure will ask you the SpagoBI external engines to be installed: you can choose those engines you need. You will be able also to choose if you want the SpagoBI example documents (that are the SpagoBIDemo example documents) to be installed or not.

Remote repository and local repository
SpagoBI core and external engines binary packs are not included into this installation file but don't worry, they will be automatically downloaded from a remote repository into a local folder of your choice (that will be referred to as 'local repository'). For later installation procedures you will be able to avoid downloads by selecting the local repository folder when required.

System requirements
Before proceeding please check the following system requirements:

  • JDK version 1.5.x
  • OpenOffice 2.0.x (for booklets creation): an environment variable named OPENOFFICE_HOME must point to OpenOffice installation directory or the PATH environment variable must contain the OpenOffice installation program folder.
  • 512 MB memory (suggested).

Server requirements
SpagoBI Platform must be installed on an existing server installed on your pc. The server can be Tomcat, JBoss or JOnAS (you can choose) and must have eXo-Portal installed. Possible servers for SpagoBI Platform 1.9.2 are:

  1. Tomcat 5.0.28 with eXo-Portal 1.1.4
    download eXoPortal-tomcat-1.1.4.zip from:
    http://forge.objectweb.org/project/download.php?group_id=151&file_id=7208
    unzip it into a local folder and the job is done.
  2. JBoss 4.0.3SP1 with eXo-Portal 1.1.2
    download jboss-4.0.3SP1.zip from:
    http://sourceforge.net/project/showfiles.php?group_id=22866&package_id=16942&release_id=365509
    download eXoPortal-jboss-1.1.2.zip from:
    http://forge.objectweb.org/project/download.php?group_id=151&file_id=6228
    then follow the installation instructions.
  3. JOnAS 4.7.4 with eXo-Portal 1.1.3
    download jonas4.7.4-tomcat5.5.15.tgz from:
    http://forge.objectweb.org/project/download.php?group_id=5&file_id=6132
    download eXoPortal-JOnAS-1.1.3.zip from:
    http://forge.objectweb.org/project/download.php?group_id=151&file_id=6838
    then follow the installation instructions.
SpagoBI installation requires overwriting some existing files on server, therefore it is suggested to install SpagoBI on a clean server.
